# CoreInBe NextGen: Classroom Monitoring System

**CoreInBe NextGen** — это абсолютно новая система мониторинга класса, переписанная с нуля для максимальной скорости и точности.

## Особенности
*   **Мгновенный запуск**: Камера открывается сразу при запуске `main.py`.
*   **Цветные боксы**: Зеленый (OK), Красный (Нарушение: Телефон).
*   **Автоматизация**: Никаких команд в Telegram, бот работает только на прием фото.
*   **Полноэкранный режим**: Окно мониторинга всегда сверху и на весь экран.

## Как запустить в PyCharm
1.  Установите библиотеки: `pip install ultralytics aiogram opencv-python`
2.  Запустите `main.py`.
3.  Нажмите 'q' для выхода из камеры.

## Как опубликовать на GitHub
1.  Создайте новый репозиторий на GitHub.
2.  Загрузите все файлы из этой папки.
3.  В описании укажите, что это система на базе YOLOv8 для мониторинга класса.

## Как использовать с Google AI Studio (Gemini 1.5 Pro)
1.  Получите API ключ на [aistudio.google.com](https://aistudio.google.com/).
2.  Используйте промпт:
    > "Ты — эксперт по безопасности в классе. Я буду передавать тебе кадры с камеры. Твоя задача: мгновенно определять наличие мобильного телефона в руках студента или признаки списывания. Если видишь нарушение, выведи JSON: {'violation': true, 'type': 'phone', 'confidence': 0.95}. Если всё чисто: {'violation': false}. Анализируй позу рук относительно головы."
3.  Вы можете интегрировать этот API в `monitor.py` для еще более умного анализа.
